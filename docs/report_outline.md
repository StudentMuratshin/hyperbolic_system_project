# План отчета

## 1. Постановка задачи

Тема: сравнительный анализ численных потоков Русанова, HLL и Roe для одномерной гиперболической системы уравнений мелкой воды.

Цель работы: реализовать конечно-объемный метод на C++ и сравнить три приближенных решателя задачи Римана по точности, устойчивости, численной вязкости и вычислительной стоимости.

## 2. Математическая модель

Уравнения мелкой воды:

```text
d/dt h + d/dx (hu) = 0,
d/dt (hu) + d/dx (hu^2 / h + g h^2 / 2) = 0.
```

Здесь `h` - высота слоя воды, `u` - скорость, `g` - ускорение свободного падения.

Консервативная форма:

```text
U_t + F(U)_x = 0,
U = (h, hu)^T.
```

## 3. Гиперболичность

Собственные значения:

```text
lambda_1 = u - sqrt(g h),
lambda_2 = u + sqrt(g h).
```

При положительной высоте `h > 0` оба значения действительны, следовательно система гиперболическая.

## 4. Численный метод

Используется конечно-объемная схема первого порядка:

```text
U_i^{n+1} = U_i^n - dt/dx * (F_{i+1/2} - F_{i-1/2}).
```

Шаг по времени выбирается из условия CFL:

```text
dt = CFL * dx / max_i(|u_i| + sqrt(g h_i)).
```

## 5. Сравниваемые потоки

### Rusanov

```text
F = (F_L + F_R) / 2 - a (U_R - U_L) / 2,
a = max(|u_L| + c_L, |u_R| + c_R).
```

### HLL

```text
S_L = min(u_L - c_L, u_R - c_R),
S_R = max(u_L + c_L, u_R + c_R).
```

При `S_L <= 0 <= S_R` используется промежуточный поток HLL.

### Roe

Поток строится через Roe-осреднение и разложение скачка по собственным векторам. Для устойчивости около звуковой точки добавлена энтропийная поправка Хартена.

## 6. Численные эксперименты

1. Распад разрыва `h_L = 2`, `h_R = 1`.
2. Сильный распад разрыва `h_L = 10`, `h_R = 1`.
3. Гладкое периодическое начальное условие `h = 1 + 0.2 sin(2 pi x)`.
4. Проверка влияния CFL.
5. Сеточная сходимость на гладком тесте.

## 7. Критерии сравнения

- ошибка `L1` по `h` и `u`;
- сохранение массы `M(t) = sum_i h_i dx`;
- ширина правого фронта в ячейках;
- число шагов и время работы;
- наличие отрицательных высот или других признаков потери устойчивости.

## 8. Выводы

В выводах стоит подчеркнуть:

- консервативная форма корректно работает с разрывными решениями;
- все три схемы сохраняют массу с малой погрешностью;
- Rusanov обычно сильнее размазывает фронты;
- HLL является устойчивым компромиссом;
- Roe дает более резкие фронты, но требует большей аккуратности.
